# Projet C - Starter

Le projet sert de fil rouge entre les seances. Le starter montre la structure
attendue : code metier separe, executable principal, tests et Makefile.

## Commandes

```bash
cd starter
make
make run
make test
make memcheck
make clean
```

## Jalons proposes

- Jalon 1 : schema des fichiers, role de chaque module, Makefile minimal.
- Jalon 2 : prototypes `.h`, structures principales, cas de test.
- Jalon 3 : implementation partielle compilable avec `make test`.
- Jalon 4 : version candidate avec rapport `make memcheck`.
